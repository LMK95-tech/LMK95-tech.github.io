Mission 01 : DHCP & Routage

Infrastucture : Orange

Projet Technique // SISR

Mise en place d'un serveur DHCP & Routage Dynamique

Cette mission consiste à automatiser l'attribution des adresses IP pour l'infrastructure Orange. L'objectif est de supprimer la gestion statique obsolète, d'éviter les doublons d'IP et de permettre une communication fluide entre les segments via RIPv2.

350 Hôtes

Capacité Adressage

01. Analyse & Sizing

Problématique

L'entreprise utilisait une configuration manuelle provoquant des conflits d'IP critiques. Il fallait une solution auto-gérée pour 350 utilisateurs.

Objectifs

  • Création de Scopes DHCP (Pools)
  • Routage dynamique RIPv2
  • Encapsulation 802.1Q

02. Implémentation Cisco CLI

Lenny@Orange_Core_Router:~

// Configuration du Pool DHCP

Router(config)# ip dhcp pool POOL_ORANGE

Router(dhcp-config)# network 192.168.2.0 255.255.255.0

Router(dhcp-config)# default-router 192.168.2.254

// Activation RIPv2

Router(config)# router rip

Router(config-router)# version 2

Router(config-router)# network 192.168.2.0
